## 🎯 核心场景

### 场景 1: 早晨出门 (7:00)
```
触发：时间 7:00
动作:
  1. TTS 播报天气 + 通勤建议
  2. TTS 播报 AI 新闻摘要
  3. TTS 播报财经新闻摘要
  4. 发送通知 (点击可查看详情)
日志：记录触发时间和执行动作
```

### 场景 2: 低电量 (<20%)
```
触发：电量低于 20%
动作:
  1. 发送通知"电量低，建议充电"
  2. TTS 播报提醒
日志：记录电量和触发时间
```

### 场景 3: WiFi 连接
```
触发：连接到家/公司 WiFi
动作:
  1. TTS 播报"已到家/公司"
  2. 自动执行家庭/办公模式
日志：记录 SSID 和连接时间
```
