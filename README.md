# OpenClaw Clients

Multi-platform client applications for OpenClaw.

## Platforms
- **Android**: Mobile client (Java/Kotlin)
- **WearOS**: Smartwatch client (Kotlin)  
- **iOS**: iPhone/iPad client (Swift)
- **Web**: Browser-based client (React/Vue)
- **Desktop**: Cross-platform desktop client (C++/Qt)

## Getting Started
```bash
git clone https://github.com/SxLiuYu/openclaw-clients.git
cd openclaw-clients
```

## Project Structure
```
openclaw-clients/
├── android/      # Android mobile client
├── wearos/       # Wear OS smartwatch client  
├── ios/          # iOS client
├── web/          # Web client
└── desktop/      # Desktop client
```
